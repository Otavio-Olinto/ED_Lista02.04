# ED_Lista02.04
## Fatorial Duplo Recursivo

Este desafio consiste em criar uma aplicação, em Java, que receba um número natural ímpar e calcule o fatorial duplo do mesmo.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. O cáculo do fatorial duplo deve ser realizado por uma função recursiva;
3. O número recebido deve ser validado fora da recursiva;
4. Declarar, via comentário, a condição de parada da função.
```
